<h1>뉴스 데이터 크롤링 With Python</h1>
<hr/>
<h2>설명</h2>
빅카인즈 또는 네이버 API를 사용하여 뉴스기사를 크롤링합니다.
크롤링한 기사는 MySQL에 저장할 수 있습니다.
<hr/>
<h2>.env 파일</h2>
네이버 API또는 MySQL 사용안할시 공란
<hr/>
```
NAVER_CLIENT_ID=[네이버 클라이언트 ID]
NAVER_CLIENT_KEY=[네이버 클라이언트 KEY]

MYSQL_URL=localhost
MYSQL_USER=root
MYSQL_PASSWORD=
MYSQL_NAME=newsdata
```
